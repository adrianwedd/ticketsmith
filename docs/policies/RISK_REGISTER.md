# Risk Register

| ID | Description | Likelihood | Impact | Mitigation | Owner |
|----|-------------|-----------|--------|-----------|-------|
| R1 | Unpatched dependency vulnerability | Medium | High | Weekly dependency scans and updates | SecurityLead |
